# SaaS Customer Churn & Retention Analytics Dashboard

🔍 Project Summary

This project delivers an end-to-end analysis of SaaS customer behavior to identify churn drivers, high-risk segments, and revenue impact.
An interactive dashboard is developed to translate data into clear, decision-ready insights for business stakeholders.

🎯 Business Objectives
Quantify overall customer churn and retention performance
Identify segments with elevated churn risk
Analyze behavioral patterns driving churn
Assess financial impact of customer loss
Recommend targeted retention strategies

📁 Data Overview
The final dataset is constructed by integrating multiple sources:

Customer accounts
Subscription plans
Feature usage metrics
Support ticket interactions
Churn event records

Processed dataset:
Merged_SaaS_Subs_cleaned_data.csv

⚙️ Technology Stack
Python (Pandas) – Data processing & analysis
Streamlit – Interactive dashboard deployment
Plotly – Dynamic, user-driven visualizations

📊 Dashboard Highlights
• Business Performance Overview

Tracks key metrics including churn rate, customer base, tenure, and revenue loss.

• Customer Drop-off Analysis

Identifies critical stages where user engagement declines.

• Churn by Plan Tier

Highlights subscription segments with higher churn exposure.

• Customer Risk Segmentation

Evaluates churn likelihood based on:

Product usage intensity
Support dependency
• Churn Timing Analysis

Examines lifecycle stages where churn is most likely to occur.

• Revenue Impact Assessment

Quantifies revenue loss across different customer segments.


📌 Key Insights
Low engagement is the primary driver of churn
Early lifecycle customers show the highest vulnerability
High support usage correlates with increased churn risk
Revenue loss is concentrated within specific plan tiers
🚀 Strategic Recommendations
Strengthen onboarding to improve early-stage engagement
Implement proactive support for high-risk customers
Prioritize retention efforts for high-value segments
Monitor behavioral signals for early churn intervention
